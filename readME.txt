---------------------
Program służy do interpretowania języka Pętlik na język C. Pętlik składa się z podstawowych
instrukcji takie jak inkrementacja, dodawanie, zerowanie i pętla. Program zawiera
funkcje optymalizujące dodawanie.
--------------------
Program is an interpreter of the "Petlik" programming language. It translates programs 
from Petlik to C. Petlik is a language with commands such as incrementations, addition,
clearing variables and while loop.
